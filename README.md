
# FRS

A faculty review system which allows students to rate the faculties on the basis of their interactions.


## Features

- Easy Student Registration
- Feedback 
- Rating
- Cross platform
- Customised Design
- Dynamic Web Pages

## Installation

Install 9down_FRS_Aviothic1.0 with npm

```bash
  npm install
```


## Deployment

To deploy this project run


```bash
  npm start
```
